# Bản đồ Kỹ năng (Skills Mapping)

Bảng này giúp Agent xác định ngay kỹ năng cần sử dụng dựa trên từ khóa trong yêu cầu của người dùng.

| Tên Skill (Key) | Mục đích sử dụng / Từ khóa kích hoạt (Value) | Đường dẫn |
| :--- | :--- | :--- |
| **React Query Standard** | Khi cần ghép API, xử lý loading/caching, Thêm/Sửa/Xóa dữ liệu (CRUD). | `Agent/Skill/react_query_standard.md` |
| **Zalo Login B2B Flow** | Khi cần làm nút "Đăng nhập bằng Zalo" kèm theo ID doanh nghiệp (SaaS). | `Agent/Skill/zalo_login_b2b_standard.md` |
| **ZMP UI Theme** | Khi cần chỉnh sửa giao diện, màu sắc, Dark mode hoặc Zalo Theme. | (Sắp có) |
| **Multi-tenancy Flow** | Khi xử lý logic liên quan đến Tenant ID, chi nhánh, phân quyền doanh nghiệp. | (Sắp có) |

---
*Ghi chú: Luôn ưu tiên dùng các kỹ năng đã được chuẩn hóa trong thư mục `Agent/Skill/`.*
